# TypeScript Setup

Welcome! We'll start with some TypeScript exercises.

Before we begin, let's make sure these dependencies are installed

1. ## Install node.js if needed: https://nodejs.org/en

You can check if it is installed with these commands

```bash
node -v
npm -v
npx -v
```

If it no work, then download with this link here: [nodejs](https://nodejs.org/en)

2. ## Install TypeScript

Globally:

```bash
npm install -g typescript
```

3. ## Install TypeScript

Globally:

```bash
npm install -g typescript
```
